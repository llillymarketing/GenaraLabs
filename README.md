# Genara Labs

Research-grade peptide supplier website built with React + Tailwind CSS.

## Stack
- React 18
- Tailwind CSS
- Framer Motion
- React Router DOM
- Base44 backend (entities, auth, integrations)

## Pages
- Home - Landing page
- Products - Peptide catalog (launching soon)
- About - Company info
- Contact - Contact form
- Cart - Shopping cart
- Checkout - Order form
- ProductDetail - Individual product page
- OrderConfirmation - Post-order page

## Legal Pages
- Terms of Service
- Privacy Policy
- Disclaimer
- Shipping Policy
- Refund Policy
